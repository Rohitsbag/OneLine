# Search

Search across all your journal entries using full-text search.

## Parameters

<ParamField query="q" type="string" required>
  Search query (2-200 characters).
</ParamField>

<ParamField query="limit" type="number">
  Maximum results (1-10). Default: 10.
</ParamField>

## How Search Works

The search uses case-insensitive substring matching with similarity scoring:

- **Exact matches** score higher than partial matches
- Results are sorted by similarity score, then by date
- Maximum 10 results per query

<Tip>
Use specific keywords for better results. For example, search for "birthday" instead of "day".
</Tip>
